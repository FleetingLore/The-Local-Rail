之前小说的名字叫 `雨声`。中午的时候我在洗澡 ，想了想，果断决定重写小说。而小说的名字后面加上 `千年` 两个字。

之前的创作，很吝啬于已有的文字内容，然后大规模地构造。这一次，我想以构造为基础，以内容为目标，指望水到渠成。

我想先完成构造，等我看完了 Mc 和花语程行的小说，再继续动笔。

为了构思形象，我学习画画有一年多了，但实力还是不行。再接再厉。

以下是创作框架。

```
+ 雨声千年
  | 小说内的空间
  + 时域设定 = 对于一定的空间，包括时代和关键事件
  + 地理设定 = 各个层级的地区的设定
  
  | 世界观的抽象构成
  + 创作日志 = 小说的基础积累，构思和写作经历
  + 思想背景 = 小说想表达的思想，以及它所支撑的小说架构

  | 世界观的具象构成
  + 社群设定 = 社会群体以及代表形象
  + 形象设定 = 包括人物形象和福瑞形象等叙事主体

  | 小说外的空间
  + 正文 = 小说的文字目标，另外包括每一篇的大纲，发展规划和注释
  + 碎片 = 无组织的内容
```
